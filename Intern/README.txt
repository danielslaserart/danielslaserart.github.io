Daniels Laser Art – Kalkulator

Enthalten:
- index.html
- style.css
- app.js
- manifest.json
- sw.js

Verwendung:
1. Den kompletten Ordner auf einen Webspace oder GitHub Pages hochladen.
2. index.html im Browser öffnen.
3. Auf Android in Chrome: Menü → Zum Startbildschirm hinzufügen.
4. Danach verhält sich die Seite fast wie eine App.

Daten:
- Materialien und Einstellungen werden lokal im Browser gespeichert.
- Unter „Sicherung“ kann eine JSON-Sicherung heruntergeladen und später wieder eingelesen werden.
